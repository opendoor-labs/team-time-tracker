PER-TEAM SHEET LOGGING · DEMO

How task data lands in the shared Google Sheet

Every task you log in the app writes one row into your team's dedicated tab. Each team has its own columns (matching the Google Doc spec), so Listings never see SD fields, HQI gets the dual ticket-link column, and so on. The TL dashboard reads directly from these tabs.

📝 One write per task

When you click Log Task in the app, a single row is appended via sheets.values.append to {Team}_Log. No SELECT *, no cross-team mixing — each team's tab stays clean.

🧩 Columns match the doc

Columns come verbatim from the Task Field Doc. Listings gets Task / Ticket Link / MARKETS / Sub Task; HQI gets both Ticket Link/Property Address AND Ticket Link; Dispatch uses Property Address/Ticket Link, etc.

🔒 Privacy by design

Try the "View as" dropdown above — switch to SD TL and watch Listings_Log / HQI_Log / etc. lock up with a 🔒 icon. The TL dashboard pulls ONLY tabs the TL is scoped to via whitelist[tl_email] → [team]. Attendance & Sessions are the only cross-team reads.
