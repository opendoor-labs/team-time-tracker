How to use this dashboard

  • Pick a Period (Today, This Week, etc.) to set the date range
  • Narrow down with Team, User, Activity, or Task filters
  • Toggle Compare to previous to see ▲/▼ vs last period
  • Pick a single user to see their personal task log
  • Click the × on any filter chip to remove it
